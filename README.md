# pnpm workspace template

A basic pnpm workspace template, default configure react code style.

## Basic

Git

- husky
- commitlint
- lint-staged

Code style

- eslint
- prettier

Code lib

- typescript
- nanoid

## Initialization

```shell
pnpm postinstall # init husky
# use latest package, Not sure it's safe.
# pnpm update
```
